toolove for MacOS

curl -sS https://toolove.falseinput.com/install.sh | sh

Installs essential missing parts of your os:

Brew
Ghostty
Nerd fonts
Starship
Mise
Aerospace
Docker
Neovim

It automatically configures .zshrc to use Starship prompt & load Mise context.

Too see changes immediately, reload your shell: source ~/.zshrc
