About init.mac

You just got a new mac from IT department. They have taken care of installing all remote control and update stuff. That's the part you don't care about.

mac.init sets up the other part.

It installs opinionated set of tools, that is, stuff that I find useful. If you don't like some of it, or would like to add more, fork it.

Installation

Run in terminal: 

curl -sS https://initmac.falseinput.com/install.sh | sh

To see changes immediately, reload your shell: 

source ~/.zshrc

Shell configuration

This installer supports only .zshrc

Starship and mise are automatically configured in the .zshrc.

This means that when you cd into dir you will see Starship prompt and Mise will load context if you used mise previously in this directory.

Default keybindings

Alt + / Lay out windows in columns
Alt + 1-10 Switch to workspace
Alt + Shift + 1-10 Move active window to a workspace
Alt + < Move focus to the window on the left
Alt + > Move focus to the window on the right
Cmd + w Close window
Cmd + W Close all windows but current
Ctrl + ` Toggle quick terminal

Aerospace keybinding are located in ~/.aerospace.toml
